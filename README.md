# 📊 Sales & Customer Insights Dashboard

## 🔍 Overview
This project presents an interactive **Power BI dashboard** designed to analyze sales performance and generate actionable business insights. The dashboard helps stakeholders understand trends, identify high-performing regions and categories, and support data-driven decision-making.

## 📊 Dashboard Structure

### 🟢 Page 1: Overview Dashboard

- KPI Metrics:
  - Total Sales
  - Total Profit
  - Total Orders
  - Profit Margin
- Sales by Region
- Profit by Category
- Monthly Sales Trend
- Key Business Insights


### 🟣 Page 2: Detailed Analysis

- Top Sub-Category Sales Analysis
- Sales vs Profit Scatter Plot
- Customer Segment Distribution
- Region-based Filtering
- Deep-dive insights for decision making


## 🚀 Features

- 📌 Interactive dashboards with slicers (filters)
- 📈 KPI tracking and performance monitoring
- 📊 Time-series analysis (monthly trends)
- 🔍 Drill-down analysis using multiple pages
- 🎯 Business insights generation


## 🛠️ Tools & Technologies

- **Power BI** (Dashboard Development)
- **Excel / CSV Dataset** (Data Source)
- **DAX** (Measures and calculations)
- **Power Query** (Data cleaning and transformation)


## 📈 Key Insights

- Central region generates the highest sales
- Technology category yields maximum profit
- Sales show a consistent upward trend over time
- Certain sub-categories have high sales but lower profit margins
- Consumer segment contributes the largest share of revenue

## 📌 How to Use

1. Download the `.pbix` file
2. Open in **Power BI Desktop**
3. Explore:
   - Use filters (Region, Category, Date)
   - Navigate between pages using buttons
   - Interact with visuals for deeper insights


## 💡 Business Use Case

This dashboard can be used by:
- Business analysts to monitor sales performance
- Managers to identify profitable regions and products
- Decision-makers to optimize strategies based on trends
